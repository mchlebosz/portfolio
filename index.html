O mnie

Nazywam się Mateusz Chlebosz, jestem uczniem klasy maturalnej w Zespole Szkół Komunikacji w Poznaniu. Dobrze czuję się w wielu dziedzinach informatyki, jednak od kilku lat moją pasją stał się frontend. W tym kierunku wciąż się rozwijam wciąż poznając nowe narzędzia i technologie. Znalazłem również angaż przy kilku projektach komercyjnych, dzięki czemu udało mi się zdobyć praktyczne doświadczenie i elementy, które mogę pokazać w portfolio. Posiadam również doświadczenie w firmach jako webdeveloper, pracując w technologiach jakich jak, wordpress, bootstrap, gulp, node czy sass. Z chęcią kontynuuję tą ścieżkę kariery wciąż szukając nowych wyzwań. Może to Twoją stronę zrobię jako następną?

Technologie

c++

5-cio letnie doświadczenie w języku C++ oraz 2-letnie doświadczenie w obiektowym C++. Znajomość biblioteki stl. Pracowałem nad projektami z wykorzystaniem SFML

Python

3-letnie doświadczenie w Pythonie. Znajomość podstaw języka, na koncie kilka mniejszych skryptów.

Javascript

2-letnie doświadczenie intensywnej pracy w JS z wykorzystaniem najnowszych standardów. Praca z wykorzystaniem jQuery.

php

2-letnie doświadczenie w pracy przy pisaniu dynamicznych stron internetowych w PHP.

bootstrap

2-letnie doświadczenie intensywnej pracy z wykorzystaniem technologii Bootstrap w wersji 4

node.js

Roczne doświadczenie z technologią Node.js, frameworkami stron oraz menadżerem npm

git

5-cio letnie doświadczenie z wykorzystaniem technologii git, głównie z pomocą serwisów GitHub oraz Bitbucket

sass

2-letnie doświadczenie w wykorzystywaniu technologii SASS przy tworzeniu skomplikowanych arkuszy stylów do stron internetowych

wordpress

2-letnie doświadczenie tworzenia stron oraz motywów do Wordpress'a, znajomość funkcji oraz wielu wtyczek
